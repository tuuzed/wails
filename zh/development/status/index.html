Linux 特定

由*_linux.go文件使用的函数的实现详细信息位于以下文件中：

  • linux_cgo.go：CGo 实现
  • linux_purego.go：PureGo 实现

CGO

默认情况下，使用 CGO 编译 Linux 端口。这会阻止轻松的交叉编译，因此同时也正在同时 开发 PureGo 实现。

Purego

可以使用以下命令编译示例：

CGO_ENABLED=0 go build -tags purego

注意：重构之后的功能目前无法正常工作。
